# LifeOS

LifeOS is a personal productivity workspace for organizing tasks, planning the day, protecting focus time, tracking habits, reviewing progress, and keeping important work visible.

Live app: [https://life-os-4e911.web.app/](https://life-os-4e911.web.app/)

## What It Does

LifeOS brings the most important parts of a personal planning system into one workspace:

- Capture tasks quickly.
- Organize tasks by project, tag, priority, due date, and energy level.
- Plan the day with Top 3 priorities and a Deep Work task.
- Add time blocks for focused planning.
- Run focus sessions and review focus history.
- Track habits and weekly consistency.
- Review the week with guided reflection questions.
- Save custom task views for different contexts.
- View simple insights and reports about task and focus patterns.
- Customize the app with themes, dark mode, accent colors, and app icons.
- Export, import, and delete personal app data from Settings.

## Why I Built It

I built LifeOS to solve a common productivity problem: tasks, focus, habits, and weekly reflection are often separated across different tools. LifeOS combines them into one calmer system so a user can answer three practical questions:

- What matters today?
- What did I actually do?
- What needs attention next?

The goal was not to create a complicated dashboard, but a personal operating system that stays useful day after day.

## Main Sections

### Dashboard

A command center showing the most important current signals: today's plan, focus progress, useful recommendations, quote, and shortcuts.

### Inbox

A place to capture loose tasks before organizing them.

### Today

A daily planning page with today's tasks, overdue tasks, Top 3 priorities, Deep Work, time blocks, quick capture, and reflection.

### Projects

A project workspace for grouping tasks by outcome, area, and status.

### Saved Views

Reusable task views for filtering by status, priority, project, tag, due date, and energy.

### Focus

A focus timer for structured work sessions, linked tasks, session notes, and focus history.

### Insights

A reporting area for understanding focus time, completed tasks, project attention, priorities, and tags.

### Habits

A habit tracker for daily completion, weekly progress, streaks, and archived habits.

### Weekly Review

A guided weekly reflection flow for reviewing wins, struggles, lessons, energy, mood, focus, habits, projects, and next-week priorities.

### Settings

Personalization, installation guidance, notification preferences, data backup, import, and account/data controls.

## How To Try It

1. Open the live app: [https://life-os-4e911.web.app/](https://life-os-4e911.web.app/)
2. Create an account with a real email address.
3. Verify the email from your inbox.
4. Add a few tasks and projects.
5. Try Today planning, Focus, Habits, and Weekly Review.
6. Open Settings to test themes, dark mode, app icons, and backup/export.

Use this flow to understand the app quickly:
